# Skill Pack: Performance Tuning

## Trigger

Use when latency, throughput, or resource usage regresses.

## Method

1. Define baseline metric and workload.
2. Profile hot path and identify top bottleneck.
3. Apply one optimization change at a time.
4. Re-measure and compare against baseline.

## Output

- Before/after metric table
- Bottleneck root cause
- Optimization tradeoff notes
