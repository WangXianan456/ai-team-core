# Skill Pack: Incident Response

## Trigger

Use during production incidents or severe release regressions.

## Procedure

1. Classify severity and impacted scope.
2. Stabilize first (mitigation or rollback).
3. Preserve evidence (logs, metrics, timeline).
4. Communicate status and ETA.
5. Produce postmortem with prevention actions.

## Output

- Incident timeline
- Mitigation actions
- Root cause and follow-up tasks
