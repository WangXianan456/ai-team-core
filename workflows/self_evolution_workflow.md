# Self Evolution Workflow

Use this workflow when the AI team upgrades itself.

## Trigger

- Weekly optimization slot
- Repeated delivery pain (cost, quality, speed)
- New role/tool requirement

## Steps

1. Create improvement task card (`TASK-9xx`) linked to an `EPIC`.
2. Define one measurable target metric.
3. Implement one scoped change only.
4. Run one real delivery task as validation.
5. Compare before/after metrics from report scripts.
6. Keep or revert based on evidence.

## Evidence Checklist

- `scripts/report.py weekly` output
- `scripts/cost_report.py weekly` output
- QA pass rate trend
