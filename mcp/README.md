# MCP Integration

This folder documents role-specific MCP connections.

## Recommended Files

- `servers.md`: MCP servers and connection info
- `permissions.md`: role-based access matrix
- `tools_catalog.md`: allowed tools and usage limits

## Example Use

- PM: docs/jira read-only servers
- Dev: code host and issue tracker, limited write
- QA: test runner and artifact readers
- Ops: deployment and monitoring tools

Store credentials outside Git.
